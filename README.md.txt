# Tweet Sentiment Analysis — Мой первый учебный проект по ML

## Описание
Это мой первый проект по машинному обучению, где я попыталась предсказывать настроение твитов: позитивное, нейтральное или негативное. Для работы использовала готовый датасет `tweet_eval` с уже размеченными данными. Я применила два классических алгоритма классификации — `LogisticRegression` и `MultinomialNB`. Проект остался сырым: я не доработала предобработку текста и настройку моделей, так как в основном изучаю анализ данных, но решила попробовать свои силы в ML.

## Технологии
- **Язык:** Python  
- **Библиотеки:** `pandas`, `scikit-learn`, `datasets`, `matplotlib`, `seaborn`  
- **Датасет:** `tweet_eval` (с Hugging Face)

## Данные
- **Размер словаря:** 10,000 слов (с использованием `TfidfVectorizer`, исключены стоп-слова, не несущие эмоционального смысла).  
- **Объём данных:** 21,000 твитов для обучения (по 7,000 на каждую категорию: негатив, нейтрал, позитив).  
- **Тестовые данные:** 12,284 твита.

## Результаты
### На тренировочных данных:
- `LogisticRegression`: точность 0.744  
- `MultinomialNB` (с параметром `alpha=0.1`): точность 0.716  

### На тестовых данных:
- `LogisticRegression`: точность 0.542  
- `MultinomialNB`: точность 0.506  

Обе модели показали признаки переобучения: разница между тренировочной и тестовой точностью довольно большая. Подробности я опишу в статье на Medium (ссылка появится после публикации).
